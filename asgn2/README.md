# Assignment 2

Open `src/asg2.html` in a browser to view the hierarchical WebGL bat scene.

Files:
- `asg2.html` contains the page layout and controls
- `src/asg2.js` contains the WebGL setup, render loop, bat hierarchy, animation, poke behavior, mouse controls, and performance display
- `lib/cuon-matrix-cse160.js` provides the `Matrix4` math utilities used for the model transforms
